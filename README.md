# MdsLearningLabRecursivite

Workshop sur la récursivité en JavaScript - Une exploration interactive des concepts de récursion, backtracking et manipulation de structures de données imbriquées.

### Cloner le projet

## Exécuter le projet

### Option 1 : Ouvrir directement dans le navigateur

Ouvrez simplement le fichier `index.html` dans votre navigateur :

```bash
# Sur macOS
open index.html

# Sur Linux
xdg-open index.html

# Ou double-cliquez sur index.html
```

### Option 2 : Serveur local avec Node.js

```bash
# Avec npx (Node.js requis)
npx http-server -p 8000
```

Puis ouvrez : http://localhost:8000

## Contenu

- **index.html** : Interface principale
- **script.js** : Exercices de récursivité (1-5)
- **data.json** : Structure de données pour les exercices
- **quiz.json** : Questions de validation
- **recursivite.md** : Présentation Marp sur la récursivité

## Exercices

1. **L'Init** : Charger `data.json` de manière asynchrone
2. **Le Décompte** : Fonction récursive de compte à rebours
3. **L'Analyseur** : Compter les pages totales dans l'arborescence
4. **Le Chercheur** : Trouver une URL par nom (backtracking)
5. **Le Builder** : Générer dynamiquement le menu HTML

**Ouvrez la console (F12) pour voir les résultats.**
